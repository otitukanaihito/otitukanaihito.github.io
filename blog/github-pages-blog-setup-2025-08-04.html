GitHub Pagesでアフィリエイトブログを構築した話

🎯 今回の目標

2025年8月4日、GitHub Pagesを使ってアフィリエイト機能付きブログサイトを構築しました。 目標は「無料で始める収益化サイト」の実現です。

📋 完成した機能

  • メインページ: アフィリエイト機能付きブログサイト
  • プライバシーポリシー: 法的要件を満たす完全版
  • ブログ記事: Obsidian活用ガイド（サンプル）
  • セキュリティ対策: セキュリティヘッダー、プライバシー保護

💰 アフィリエイト機能

おすすめツールセクションとして、以下の3つのツールを紹介しています：

  • Obsidian: 第二の脳構築ツール
  • ChatGPT Plus: AI活用の最前線
  • Notion: チーム協働のための万能ツール

🔧 技術仕様

  • フレームワーク: 静的HTML + Tailwind CSS
  • ホスティング: GitHub Pages（無料）
  • ドメイン: https://otitukanaihito.github.io
  • 分析: Google Analytics 4対応
  • セキュリティ: HTTPS、セキュリティヘッダー

🚀 デプロイ手順

1. GitHubリポジトリ作成

  • リポジトリ名: otitukanaihito.github.io
  • 可視性: Public
  • ライセンス: MIT License

2. Git設定

git config --global user.name "your-username"
git config --global user.email "your-email@example.com"
git init
git add .
git commit -m "Add complete blog site with affiliate features"
git remote add origin https://github.com/your-username/your-repo-name.git
git push -u origin main

セキュリティ注意: 実際のメールアドレスやユーザー名は公開しないでください。上記は例です。

3. GitHub Pages設定

  • Source: Deploy from a branch
  • Branch: main
  • URL: https://otitukanaihito.github.io

🔍 SEO最適化

Google Search Consoleでの所有権確認を完了し、以下のSEO対策を実装しました：

  • サイトマップ: XML形式で正常動作・送信完了
  • robots.txt: 検索エンジン対応完了
  • ページ構造: 3ページ（メイン・ブログ・プライバシー）最適化済み
  • 更新頻度設定: 週次・月次・年次で適切に設定

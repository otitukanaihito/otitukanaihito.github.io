サイト訪問者分析の実装方法

1. Google Analytics 4の設定

ステップ1: GA4プロパティの作成

  1. Google Analyticsにログイン
  2. 「管理」→「プロパティ作成」
  3. プロパティ名: 「オチつかないカンパニー」
  4. データストリーム: Webサイト
  5. 測定IDを取得（例: G-XXXXXXXXXX）

ステップ2: トラッキングコードの実装

<!-- Google Analytics 4 -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    gtag('js', new Date());
    gtag('config', 'G-XXXXXXXXXX', {
        'anonymize_ip': true,
        'cookie_flags': 'SameSite=None;Secure'
    });
</script>

ステップ3: カスタムイベントの設定

// ページビューイベント
gtag('event', 'page_view', {
    'page_title': document.title,
    'page_location': window.location.href,
    'custom_parameter_1': 'blog_article'
});

// 訪問者タイプの追跡
gtag('event', 'user_engagement', {
    'engagement_time_msec': 1000,
    'custom_parameter_1': 'returning_visitor'
});

2. リアルタイム訪問者カウンター

カスタムカウンターの実装

<!-- 訪問者カウンター -->
<div id="visitor-counter" class="bg-indigo-600 text-white p-4 rounded-lg">
    <h3 class="text-lg font-semibold mb-2">現在の訪問者数</h3>
    <div class="text-3xl font-bold" id="current-visitors">--</div>
    <div class="text-sm mt-2">総訪問者数: <span id="total-visitors">--</span></div>
</div>

<script>
// 訪問者カウンターの更新
function updateVisitorCounter() {
    // GA4のリアルタイムデータを取得
    gtag('event', 'get_visitor_count', {
        'event_callback': function() {
            // カウンターを更新
            document.getElementById('current-visitors').textContent = 
                Math.floor(Math.random() * 50) + 1; // 仮のデータ
        }
    });
}

// 5秒ごとに更新
setInterval(updateVisitorCounter, 5000);
updateVisitorCounter();
</script>

3. 訪問者分析ダッシュボード

分析可能なデータ

リアルタイムデータ

  • • 現在のアクティブユーザー数
  • • ページビュー数
  • • イベント数
  • • ユーザーの地理的分布

履歴データ

  • • 日別・月別訪問者数
  • • 人気ページランキング
  • • 検索キーワード
  • • デバイス・ブラウザ情報

カスタムレポートの作成

// カスタムレポートの設定
gtag('event', 'custom_report', {
    'report_type': 'visitor_analysis',
    'date_range': 'last_30_days',
    'metrics': ['users', 'pageviews', 'sessions'],
    'dimensions': ['date', 'page_title', 'device_category']
});

4. プライバシーとコンプライアンス

プライバシー保護の重要性

• IPアドレスの匿名化

• 個人情報の保護

• GDPR/CCPA準拠

• プライバシーポリシーの更新

実装済みのプライバシー対策

  • IP匿名化: 訪問者のIPアドレスを匿名化
  • セキュアCookie: セキュリティ設定付きCookie
  • データ保持期間: 26ヶ月で自動削除
  • オプトアウト: 訪問者の選択権を尊重

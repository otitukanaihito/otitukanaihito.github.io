# drafts/ - 予約投稿フォルダ

ここに置いたHTMLファイルは、ファイル名の日付になると自動的に `blog/` へ移動・公開されます。

## ファイル命名規則

```
{スラッグ}-YYYY-MM-DD.html
```

例:
- `new-article-2026-03-24.html` → 2026年3月24日（月）に公開
- `affiliate-review-2026-04-07.html` → 2026年4月7日（月）に公開

## 運用フロー

1. 記事を作成 → このフォルダに保存
2. GitHub にプッシュ
3. 毎週月曜 8:00 JST に自動チェック → 公開日を迎えた記事が `blog/` に移動

## 手動で今すぐ公開したいとき

GitHub の Actions タブ → 「予約投稿の自動公開」→ 「Run workflow」ボタンで即時実行できます。
